# Fizik Hesap Makinesi

## Proje Açıklaması
Fizik Hesap Makinesi, fiziksel hesaplamaları kolaylaştırmak için tasarlanmış bir araçtır. Kullanıcıların çeşitli fiziksel formülleri kullanarak hesaplamalarını yapmalarına imkan tanır.

## Özellikler
- Kullanıcı dostu arayüz
- Temel fizik formüllerinin entegrasyonu
- Hızlı ve güvenilir hesaplamalar
- Farklı birim sistemlerini destekleme

## Kurulum
1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/mhrrm3641-dot/Fizik_hesap_makines.git
   ```
2. Gerekli bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

## Kullanım
1. Uygulamayı başlatın:
   ```bash
   npm start
   ```
2. Hesaplamak istediğiniz fiziksel formülü seçin ve gerekli verileri girin.

## Geliştirme
Proje üzerinde geliştirme yapmak isterseniz:
